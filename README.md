# Venda de Veículos

Este é um projeto de análise de dados de anúncios de vendas de veículos, desenvolvido com Streamlit para criar um aplicativo web interativo.

## Funcionalidades

- Visualiza dados através de histogramas e gráficos de dispersão.
- Interface interativa para explorar o conjunto de dados de veículos.
